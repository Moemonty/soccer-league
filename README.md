## Soccer League

## Ruby Version
Built using Ruby 3.2.2

## Depenencies:

Install Rspec
`gem install rspec`
---

Run Rspec Init
`rspec --init`

Install `Ruby 3.2.2`
Can install with rbenv
`https://github.com/rbenv/rbenv`

---


### To run:

Run via terminal stdin:

`ruby app.rb < sample-input.txt`

Run via an argument:

`ruby app.rb sample-input.txt`
